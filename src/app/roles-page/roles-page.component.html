Access Levels

Visitor

This access level is applied to anyone visiting the website without logging in.

Restricted

This access level is applied to anyone who has just signed up. This can be used to restrict items until it has been confirmed that the person who signed up should have access to them. All members with this access level will automatically be given this Discord role:
@{{role.name}}

Member

This level should be granted to anyone who is participating in the team. It allows them to participate in the Discord server. People with this access level should be given read-only access to most data as well as write access to data for their particular team. All members with this access level will automatically be given this Discord role:
@{{role.name}}

Leader

This level should be reserved for club officers and mentors. It allows unlimited priviledges on the Discord server. All members in this role will automatically receive the same Discord role as members.
